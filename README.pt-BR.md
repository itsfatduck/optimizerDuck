## O que o optimizerDuck faz e por que é importante

O Windows funciona muito bem de fábrica. No entanto, uma instalação nova também inclui serviços em segundo plano, telemetria, aplicativos pré-instalados e tarefas agendadas que continuam rodando mesmo que você nunca as use. Tudo isso consome recursos do sistema como CPU, RAM, disco e rede em segundo plano.

Ao mesmo tempo, muitas configurações que podem melhorar o desempenho, reduzir a latência ou proporcionar uma experiência mais suave não estão habilitadas por padrão.

O optimizerDuck reúne tudo em um só lugar. Ele ajuda você a otimizar o Windows, remover componentes desnecessários, personalizar configurações do sistema e gerenciar recursos comuns do Windows sem precisar mexer no Registro, Política de Grupo ou PowerShell.

Ele também inclui ferramentas de gerenciamento integradas, permitindo que você veja o que está rodando, remova o que não precisa e restaure alterações quando necessário.

**[Por que otimizar o Windows?](#por-que-otimizar-o-windows-em-vez-de-apenas-atualizar-seu-hardware)**

### Otimizações do Sistema

Mais de 30 ajustes em 6 categorias, cada um com uma descrição clara e classificação de risco para você saber exatamente o que cada alteração faz antes de aplicá-la.

| Categoria | O que cobre |
| :----------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Desempenho** | Ajuste de host de serviço baseado na sua RAM, ajustes de prioridade de processos, redução de latência do teclado e ajustes do agendador multimídia para jogos mais fluidos |
| **Privacidade** | Desativar telemetria do Windows, relatórios de erros, ID de anúncio, rastreamento de localização, Cortana, Copilot e sugestões de entrega de conteúdo |
| **GPU** | Ajustes de registro específicos por fabricante para GPUs AMD, NVIDIA e Intel, cobrindo estados de energia, clock gating e latência de exibição |
| **Energia** | Desativar hibernação e inicialização rápida, desligar a suspensão seletiva de USB, instalar um plano de energia personalizado de alto desempenho e desativar a limitação de energia (power throttling) |
| **Bloatware e Serviços** | Bloquear o comportamento de reinstalação de aplicativos OEM e ajustar os tipos de inicialização para mais de 200 serviços do Windows |
| **Experiência do Usuário**| Remover atrasos de exibição de menus, desativar efeitos visuais como animações da barra de tarefas e transparência para uma sensação de maior rapidez |

> [!NOTE]
> As otimizações apresentadas aqui são pesquisadas de ferramentas conhecidas com grande base de usuários, nada é gerado por IA ou adicionado cegamente. Cada ajuste é escolhido pelo seu impacto no mundo real.

### Personalização

Sem necessidade de vasculhar o registro, apenas interruptores, menus suspensos e campos numéricos apresentados em um só lugar. Organizado em quatro categorias:

- **Área de Trabalho**: Exibir ou ocultar ícones (Este Computador, Lixeira, Rede, Arquivos do Usuário, Painel de Controle), remover setas de atalhos
- **Preferências**: Alinhamento da barra de tarefas, widgets, botões Visão de Tarefas e Finalizar Tarefa, segundos no relógio, modo escuro, extensões de arquivos, arquivos ocultos, histórico da área de transferência, visualização compacta, assistente de alinhamento, caixas de seleção de itens, menu de contexto clássico e pesquisa do Bing
- **Jogos**: Modo de Jogo, Game Bar, gravação em segundo plano, aceleração do mouse, otimizações de tela inteira, agendamento de GPU acelerado por hardware
- **Sistema**: Ativar Num Lock na inicialização

### Ferramentas Integradas

| Ferramenta | O que faz |
| :-------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------- |
| **Painel do Sistema** | Visualize os detalhes de sua CPU, RAM, GPU, unidades de armazenamento e SO em um único painel |
| **Gerenciador de Inicialização** | Veja todos os aplicativos e tarefas iniciados na inicialização, ative ou desative-os e abra seus locais de arquivo |
| **Tarefas Agendadas** | Navegue, execute, pare, ative, desative ou exclua tarefas agendadas do Windows |
| **Limpeza de Disco** | Escaneie e limpe arquivos temporários, cache do sistema, sobras do Windows Update, prefetch, miniaturas, lixeira, despejos de memória e instalações antigas do Windows |
| **Removedor de Bloatware** | Lista todos os pacotes AppX removíveis com selos de risco (Seguro, Cuidado, Desconhecido) para que você escolha o que remover |

### Por que otimizar o Windows em vez de apenas atualizar seu hardware?

A maneira mais rápida de melhorar o desempenho é sempre atualizar seu hardware. Uma CPU melhor, GPU, mais RAM ou um SSD mais rápido geralmente proporcionam melhorias muito maiores do que apenas ajustes de software.

Mas o hardware é apenas uma parte do cenário.

O Windows é projetado para centenas de milhões de PCs com hardwares, cargas de trabalho e usuários diferentes. Por isso, a Microsoft distribui o Windows com configurações que priorizam compatibilidade, estabilidade, vida útil da bateria e facilidade de uso em vez do desempenho máximo.

Como resultado, uma instalação padrão do Windows inclui muitos serviços em segundo plano, tarefas agendadas, telemetria, recursos de economia de energia e aplicativos pré-instalados que muitos usuários podem nunca precisar.

Isso é especialmente verdadeiro após reinstalar o Windows. Mesmo depois de instalar todos os drivers e atualizações, muitas configurações úteis relacionadas ao desempenho permanecem intactas, enquanto componentes desnecessários em segundo plano continuam rodando.

Otimizar o Windows é simplesmente reduzir a sobrecarga desnecessária para que seu PC possa gastar mais recursos no que realmente importa: jogos, programação, criação de conteúdo ou trabalho do dia a dia.

Isso não vai magicamente dobrar seu FPS, mas pode reduzir a atividade em segundo plano, diminuir a latência do sistema, melhorar a capacidade de resposta e ajudar seu hardware a funcionar de forma mais consistente.

Esta é também uma das razões pelas quais muitas pessoas recomendam o Linux para obter melhor desempenho e menor sobrecarga do sistema. Se você prefere ficar no Windows por sua compatibilidade de software e experiência familiar, o optimizerDuck ajuda você a obter a melhor experiência possível sem trocar de sistema operacional.

### Quando devo otimizar o Windows?

A melhor hora é **após instalar um Windows novo**.

Recomendamos esta ordem:

1. Instale o Windows.
2. Instale todos os drivers de hardware (chipset, GPU, rede, áudio, etc.).
3. Execute o **Windows Update** até não haver mais atualizações disponíveis.
4. Instale todas as **atualizações opcionais**.
5. Abra a **Microsoft Store** e atualize todos os aplicativos integrados.
6. Instale os softwares e jogos que você usa normalmente.
7. Aplique as otimizações de sua preferência no optimizerDuck.
8. Use as ferramentas integradas do optimizerDuck para remover aplicativos desnecessários, limpar o Windows e gerenciar programas de inicialização.

Seguir esta ordem ajuda a evitar que atualizações posteriores do Windows ou drivers sobrescrevam suas otimizações.

> [!TIP]
> Muitos usuários avançados optam por desativar temporariamente as atualizações automáticas do Windows após concluir a configuração. Atualizações grandes do Windows podem restaurar configurações padrão, reinstalar certos componentes ou desfazer algumas otimizações. Se você fizer isso, lembre-se de reativar periodicamente o Windows Update para instalar atualizações importantes de segurança antes de desativá-lo novamente.

> [!NOTE]
> Todas as otimizações fornecidas pelo optimizerDuck também podem ser realizadas manualmente. O objetivo do optimizerDuck é simplesmente tornar a otimização do Windows mais fácil, segura e conveniente.

## Segurança

Alterar as configurações do sistema traz riscos. O optimizerDuck é construído em torno da reversibilidade e do controle do usuário.

Consulte a [Política de Privacidade](./PRIVACY.md) para detalhes sobre nossas práticas de dados.

- **Backups automáticos**: Cada alteração grava um arquivo de reversão em uma pasta local. Você pode restaurar ajustes individuais ou reverter tudo
- **Reversão com um clique**: Desfaça qualquer otimização aplicada a partir da interface do usuário com um único clique
- **Classificações de risco**: Cada ajuste é rotulado como Seguro, Moderado ou Arriscado com base em seu impacto potencial
- **Sem padrões aplicados**: Nada é executado até que você selecione. A ferramenta não ativa nada por conta própria
- **Prompt de ponto de restauração**: Antes da primeira otimização, o aplicativo sugere a criação de um ponto de restauração do Windows

## Perguntas Frequentes (FAQ)

> [!TIP]
> Visite as [Perguntas frequentes em nosso site](https://optimizerduck.vercel.app/docs/faq/general) para mais perguntas e respostas.

### O optimizerDuck é seguro para usar?

Sim. O optimizerDuck é totalmente **código aberto** (GPL v3), o que significa que qualquer pessoa pode inspecionar, auditar ou compilar o próprio código-fonte. Cada versão é compilada automaticamente pelo **GitHub Actions** a partir do código-fonte público; sem modificações ocultas, sem binários não assinados injetados após a compilação. Se preferir, você pode clonar o repositório e compilar o `.exe` você mesmo com um simples `dotnet build`.

O aplicativo **não** coleta nenhuma telemetria, dados de uso ou informações pessoais. Consulte a [Política de Privacidade](./PRIVACY.md).

### O optimizerDuck realmente melhora o desempenho, reduz a latência ou acelera minha rede?

Pode ajudar. Cada otimização no optimizerDuck é **pesquisada de ferramentas conhecidas, guias da comunidade e recomendações de fabricantes de hardware**, nada é gerado por IA, adicionado cegamente ou inventado. Cada ajuste aborda uma configuração real que o Windows configura de forma conservadora por padrão (por exemplo, agrupamento de hosts de serviço, estados de energia da GPU, limitação de rede, agendamento de processos).

Não há hacks falsos de registro aqui, cada alteração tem um propósito documentado e impacto real no mundo real respaldado por testes da comunidade e documentação do fabricante.

### Por que o Windows SmartScreen / Defender sinaliza o download?

O optimizerDuck não possui assinatura de código digital porque certificados de assinatura de código são caros para projetos de código aberto. Quando o Windows encontra um executável não assinado baixado da internet, o SmartScreen exibe um aviso por padrão. Isso é normal e **não** significa que o arquivo não é seguro.

Para prosseguir, clique em **"Mais informações" > "Executar assim mesmo"**. Se ainda estiver preocupado:

- Compile o `.exe` você mesmo a partir do [código-fonte](https://github.com/itsfatduck/optimizerDuck)
- Envie o binário para sandboxes online como o ANY.RUN para verificação independente

### Posso reverter as alterações se algo der errado?

Sim. Cada otimização cria um arquivo de reversão antes de ser aplicada. Você pode desfazer ajustes individuais ou reverter tudo a partir da interface do usuário com um clique. O aplicativo também sugere a criação de um ponto de restauração do sistema Windows antes de sua primeira otimização.

### Isso funciona no Windows 10 e Windows 11?

Sim. O optimizerDuck suporta **Windows 10 (x64)** e **Windows 11 (x64)**.

### Preciso de privilégios de administrador?

Sim. O optimizerDuck modifica as configurações do sistema e o registro do Windows, por isso requer privilégios de administrador para ser executado.

### O optimizerDuck coleta meus dados?

Não. O aplicativo contém zero telemetria, análise ou funcionalidades de rastreamento de uso. Ele roda inteiramente offline e não envia dados para lugar nenhum.

### Por que o Gerenciador de Tarefas mostra 100% de CPU após aplicar o plano de energia? ([#29](https://github.com/itsfatduck/optimizerDuck/issues/29))

Um bug conhecido de exibição do Gerenciador de Tarefas acionado por planos de energia não padrão, que relata incorretamente 100% de CPU em alguns sistemas enquanto a carga real está normal. É apenas visual, **não** afeta o desempenho real nem causa superaquecimento. Se indesejado, basta desativar esta otimização.

## Solução de problemas

> [!TIP]
> Visite a [página de solução de problemas](https://optimizerduck.vercel.app/docs/faq/troubleshooting) para obter orientações mais detalhadas e soluções para problemas conhecidos.

### O aplicativo não inicia ou trava ao abrir

Certifique-se de estar executando como **Administrador**. O optimizerDuck requer privilégios elevados. Se ainda travar, baixe a versão mais recente do [Releases](https://github.com/itsfatduck/optimizerDuck/releases/latest); uma versão antiga pode ser incompatível com sua versão do Windows.

### As alterações não parecem surtir efeito após a aplicação

Algumas otimizações exigem uma **reinicialização do sistema** para serem aplicadas completamente. Se uma configuração não funcionar após a reinicialização, tente aplicá-la novamente ou verifique a seção de reversão para confirmar se a alteração foi salva.

### O arquivo de reversão está ausente ou corrompido

Os arquivos de reversão são armazenados em `%LocalAppData%\optimizerDuck\Revert\`. Se um arquivo for excluído acidentalmente ou corrompido, você pode restaurá-lo a partir de um backup ou criar um **Ponto de Restauração do Sistema** antecipadamente como garantia.

### O Windows Update redefine minhas configurações

As atualizações de recursos do Windows ocasionalmente redefinem certos valores do registro e configurações de serviços para os padrões. Basta reaplicar suas otimizações anteriores no aplicativo após uma grande atualização.

### Encontrei um bug / Quero solicitar um recurso

Abra uma [issue](https://github.com/itsfatduck/optimizerDuck/issues) no GitHub com o máximo de detalhes possível: sua versão do Windows, quais otimizações você aplicou e o que deu errado. Solicitações de recursos também são bem-vindas.

## Detalhes Técnicos

- **Framework**: WPF no .NET 10, usando a biblioteca WPF UI para design Fluent
- **Sistema de reversão**: Quatro tipos de etapas de reversão (Registro, Serviço, Tarefa Agendada, Shell) com estado persistido em JSON e E/S de arquivos thread-safe
- **Temas**: Modos Escuro (padrão), Claro e Alto Contraste com suporte a efeito Mica
- **Sem instalador**: Executado como um único .exe, sem necessidade de instalação
- **Sistema de backup**: Backup local baseado em pasta para cada alteração, com restauração em um clique
- **Descoberta**: Categorias de otimização e personalização são descobertas automaticamente via reflexão + atributos personalizados, sem necessidade de registro manual
- **Sem telemetria**: O aplicativo não coleta nenhum dado do usuário
