## 快速开始

1. 前往 **GitHub 发布页** 下载：https://github.com/itsfatduck/optimizerDuck/releases/latest
2. 直接运行 `.exe` 程序，**无需安装**
3. 应用优化后重启电脑即可生效

> [!TIP]
> 修改系统设置前，建议**手动创建系统还原点**。

> [!NOTE]
> 目前支持英语、越南语、繁体中文（感谢贡献者 [@abc0922001](https://github.com/abc0922001)）、简体中文（感谢贡献者 [@wcxu21](https://github.com/wcxu21)）。

## 功能亮点

内置 6 大分类、超过 35 项系统优化，附带多款实用系统工具；
所有优化均标注风险等级，支持一键还原更改。

|     | 功能模块                | 作用说明                                                                 |
| :-- | :---------------------- | :----------------------------------------------------------------------- |
| 🚀  | **性能与电源优化**      | 提升系统运行速度、禁用冗余服务、调校电源配置                             |
| 🛡️  | **隐私安全防护**        | 关闭遥测数据、小娜、Copilot、系统广告与行为追踪，夺回个人数据控制权       |
| 🎮  | **显卡专项调校**        | 针对 AMD、NVIDIA、Intel 显卡专属优化                                     |
| 🧩  | **系统功能开关**        | 一键启用或关闭各类 Windows 原生功能                                      |
| 🧹  | **磁盘清理**            | 清理临时文件、系统缓存、旧版 Windows 安装包，快速回收磁盘空间            |
| 🗑️  | **预装软件卸载**        | 卸载系统自带冗余应用，附带风险等级提示，安全可控                         |
| ⚡  | **启动项管理**          | 管控开机自启应用，保留所需、禁用无用项                                   |
| 📋  | **计划任务管理**        | 查看、手动运行、停止或删除 Windows 计划任务，全面掌控系统后台任务        |
| 💻  | **系统信息面板**        | 一键查看 CPU、内存、显卡、存储、系统版本等硬件信息                       |
| 🔄  | **优化一键还原**        | 误优化无需重装，点击即可撤销任意更改                                     |
| 🎨  | **简洁美观界面**        | 支持深色/浅色/高对比度主题，采用现代化流畅设计                           |
| 🌐  | **多语言支持**          | 英语、越南语、繁体中文、简体中文                                         |
| 🔒  | **纯离线运行**          | 无遥测、无广告、无数据收集，所有操作均在本地完成                         |

## 优化分类

内置六大优化类别，每项均配有详细说明与风险评级：

- **性能优化**：服务主机调校、进程优先级、键盘响应、游戏专项优化
- **安全与隐私**：关闭遥测、错误报告、小娜、Copilot、广告推送、位置追踪
- **显卡优化**：适配 AMD、NVIDIA、Intel 全系显卡专属调校
- **电源管理**：休眠开关、快速启动、USB 节能、自定义电源方案
- **冗余软件与服务**：卸载预装应用、精细调校 200+ 项 Windows 系统服务
- **使用体验**：资源管理器提速、视觉特效、菜单延迟等个性化微调

## 系统功能开关

不仅可以一键应用系统优化，还能可视化开关各类 Windows 隐藏功能，
无需翻找注册表、无需网上搜教程，所有选项直观易用。

- **桌面设置**：显示/隐藏此电脑、回收站、网络、用户文件夹、控制面板；去除快捷方式小箭头
- **任务栏**：图标居中、小组件、任务视图、右键结束任务、任务栏时钟显示秒数、关闭开始菜单必应搜索
- **外观**：全局深色模式适配
- **资源管理器**：显示文件扩展名、显示隐藏文件、剪贴板历史、紧凑布局、窗口吸附、文件复选框、经典右键菜单
- **游戏优化**：游戏模式、Xbox 游戏栏、后台录制、鼠标加速度、全屏优化、GPU 硬件加速调度
- **系统设置**：开机自动开启数字小键盘

## 内置工具

- **系统面板**：实时硬件信息总览
- **启动项管理**：管控开机自启程序，支持筛选、搜索、启用/禁用
- **计划任务**：完整的系统计划任务管理能力
- **磁盘清理**：一键扫描、勾选、清理垃圾文件，简单高效
- **预装软件卸载**：按风险等级列出系统自带应用，筛选后安全卸载

## 安全保障

optimizerDuck 全程把安全放在首位。
我们深知修改系统配置存在风险，因此内置多重安全防护机制。

隐私相关承诺详见：[隐私政策](./PRIVACY.md)

- **自动备份**：每一项修改都会在本地备份，可单独还原或整体恢复
- **一键还原**：操作失误随时撤销，一键恢复原有配置
- **风险分级**：每项优化标注 低/中/高 风险等级，提前评估影响
- **不强制优化**：所有优化均由用户手动选择，默认不启用任何改动
- **建议创建还原点**：应用优化前主动提示创建系统还原点，双层保障
