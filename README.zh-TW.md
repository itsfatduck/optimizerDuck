## 快速開始

1. 從 **GitHub** 下載 → https://github.com/itsfatduck/optimizerDuck/releases/latest
2. 執行 `.exe`（無需安裝）
3. 應用更改後重新啟動您的電腦

> [!TIP]
> 在進行系統更改之前，請務必建立**還原點**。

> [!NOTE]
> 支援英文、越南文和繁體中文（感謝我們的貢獻者！ [@abc0922001](https://github.com/abc0922001)）。想加入您的語言嗎？[在此貢獻](./.github/CONTRIBUTING.md)。

## 亮點

超過 35 種優化，分為 6 個類別，並內建工具來管理您的系統。所有功能都帶有清晰的風險指示和一鍵還原。

|     | 功能                 | 作用                                                                 |
| :-- | :---------------------- | :------------------------------------------------------------------------ |
| 🚀  | **性能與電源** | 加速系統、禁用臃腫服務、調整電源設置。        |
| 🛡️  | **隱私保護**  | 關閉遙測、Cortana、Copilot、廣告和追踪。取回您的數據。 |
| 🎮  | **GPU 調整**          | 廠商特定的調整。AMD、NVIDIA 或 Intel。                            |
| 🧩  | **功能切換**     | 輕鬆啟用或禁用各種 Windows 功能。                        |
| 🧹  | **磁碟清理**        | 清理臨時文件、快取、舊的 Windows 安裝。回收磁碟空間。  |
| 🗑️  | **臃腫軟體卸載**   | 刪除預裝的 Windows 應用。風險指示助您決定。       |
| ⚡  | **啟動管理**     | 控制啟動時執行的應用。保留您需要的，禁用其餘的。     |
| 📋  | **排程工作**     | 查看、執行、停止或刪除 Windows 排程工作。完全控制。         |
| 💻  | **系統儀表板**    | 一目了然查看您的 CPU、RAM、GPU、儲存和作業系統資訊。                 |
| 🔄  | **還原更改**      | 犯了錯誤？一鍵撤銷任何更改。                             |
| 🎨  | **簡潔介面**            | 深色、淺色或高對比主題。現代 Fluent 設計。                |
| 🌐  | **多語言**      | 英文、越南文、繁體中文。                                 |
| 🔒  | **離線執行**        | 無遙測、無廣告、無數據收集。一切都在本地執行。        |

## 優化類別

內建 6 個類別的優化，每個類別都有清晰的解釋和風險等級：

- **性能**：服務主機調整、優先級調整、鍵盤速度、遊戲調整。
- **安全與隱私**：遙測、錯誤報告、Cortana、Copilot、廣告、位置追踪。
- **GPU**：針對 AMD、NVIDIA 或 Intel GPU 的廠商特定調整。
- **電源管理**：休眠、快速啟動、USB 節電、自定義電源計劃。
- **臃腫軟體與服務**：刪除預裝應用。精細調整 200 多個 Windows 服務。
- **使用者體驗**：檔案總管速度、視覺效果、選單延遲。

## 功能切換

您不僅可以應用系統優化，還可以輕鬆啟用或禁用各種 Windows 功能。無需挖掘註冊表或在 Google 上無休止地搜尋。一切都可以在簡潔、直觀且使用者友好的介面中完成。

- **桌面**：顯示/隱藏圖示（本機、資源回收筒、網路、使用者文件、控制台），刪除捷徑箭頭。
- **工作列**：居中對齊、小組件、工作檢視按鈕、結束工作按鈕、在時鐘中顯示秒數、禁用 Bing 搜尋。
- **外觀**：應用程式和系統的深色模式。
- **檔案總管**：副檔名、隱藏文件、剪貼簿歷史、精簡模式、貼齊輔助、項目選取框、傳統右鍵選單。
- **遊戲**：遊戲模式、遊戲列、背景錄製、滑鼠加速、全螢幕優化、硬體加速 GPU 排程。
- **系統**：啟動時自動啟用 Num Lock。

## 內建工具

- **儀表板**：實時硬體概覽。
- **啟動管理**：控制啟動應用。過濾、搜尋、啟用/禁用。
- **排程工作**：完整的任務管理。
- **磁碟清理**：掃描、選擇、清理。就這麼簡單。
- **臃腫軟體卸載**：帶有風險等級的列表。過濾並安全刪除。

## 安全

optimizerDuck 在設計時充分考慮了安全性。我們深知修改系統設置可能存在風險，因此我們實施了多重保障措施。

有關我們對使用者隱私和數據安全承諾的詳細資訊，請參閱 [隱私政策](./PRIVACY.md)。

- **備份一切**：每次更改都備份在本地文件夾中，允許您還原單個調整或整個系統。
- **一鍵還原**：犯了錯誤？只需點擊“還原”即可撤銷您所做的最後一次更改。
- **風險等級**：在應用每個調整之前查看其風險等級。我們根據潛在影響將其分為低、中、高風險。
- **不強制更改**：您選擇要應用的優化。預設情況下不啟用任何內容。
- **建議還原點**：始終建議在應用更改之前建立 Windows 還原點，以增加一層安全性。
